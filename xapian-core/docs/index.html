Documentation

Several pieces of documentation are available.

We suggest you start by reading the Introduction, which should take you from the point of downloading the code, to getting an example application compiled and running.

For a quick introduction to our software, including a walk-through example of an application for searching through some data, read the Quickstart document.

Read the document on Information retrieval for a (reasonably mathematical) explanation of what it is we're actually trying to calculate, and why.

The User manual details the API which Omsee provides to programmers.

Much useful documentation is automatically extracted from the source code. Full documentation of the API is available for users, and documentation of the internals is available for developers working on Omsee itself. Finally, there is a short document about the Directory structure which is automatically generated from the source code.

If you want to read more about the subject of Information retrieval, there are several good books on the subject. We have found both "Readings in Information Retrieval" (published by Morgan Kaufmann, edited by Karen Sparck Jones and Peter Willett), and "Managing Gigabytes" (also published by Morgan Kaufmann, written by Ian H. Witten, Alistair Moffat and Timothy C. Bell) to be extremely useful.
