Documentation

Several pieces of documentation are available.

We suggest you start by reading the Introduction, which should take you from the point of downloading the code, to getting an example application compiled and running.

Read the document on Information retrieval for a (reasonably mathematical) explanation of what it is we're actually trying to calculate, and why.

The document on Code structure is automatically generated from the source code. You may initially find it much easier to read this than grub around in the code.

There is currently no User manual, but since we don't yet have a defined API we don't consider this a serious problem. Such a manual will be produced in the fullness of time.

If you want to read more about the subject of Information retrieval, there are several good books on the subject. We have found "Readings in Information Retrieval" (published by Morgan Kaufmann, edited by Karen Sparck Jones and Peter Willett), to be extremely useful.
